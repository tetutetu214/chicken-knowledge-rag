# 概要

宇都宮大学・杉田昭栄教授による鳥類視覚受容機構の解説論文（バイオメカニズム学会誌、2007年）。鳥類が哺乳類より優れた視覚を持つ仕組みを、眼球の構造・レンズ系の調節機構・網膜と視細胞・視物質と油球・紫外線受容・中心窩と側頭窩・神経節細胞数まで、解剖学的に体系的にまとめている。**ニワトリ（Gallus gallus）に関する具体的な記述が随所にある**ため、鶏の視覚特性を扱う一次ソースとして有用。

---

## 1. 鳥類の視野の特徴

鳥類の視野は一般的に広く、ヒトより総視野が大きい。

| 動物 | 片眼視野 | 両眼視野 | 総視野 |
|---|---|---|---|
| ヒト | — | — | 200° |
| ハト | 169° | 22° | 316° |
| ヤマシギ | — | 4.5° | 359° |
| フクロウ | — | 42° | ヒトと大差なし |

**鳥類の視野が広いのは、頭部の側方に眼球がついているため**。ハトは両眼視野が22°と狭いが、頭部の動きを利用して視野を広く・安定化させている。鳥類の頸椎は哺乳類より約2倍多く（**哺乳類7個、鳥類13〜14個**）、頸の可動範囲が広く柔軟。ハトの歩行で見られる「頭部が前後に動く動き」は、眼球の位置を絶えず安定させ対象物に焦点を合わせながら移動するための動きと考えられている。

フクロウのように素早く動く捕食者を追う必要のある鳥は、眼球が頭部前面についており両眼視野が他の鳥より広く42°で立体視に優れる反面、総視野はヒトと大差ない。

---

## 2. 鳥類の眼球の大きさ

鳥類の眼球は、頭部の器官ではほぼ脳と同じくらい、あるいはそれ以上の大きさ。眼球の長軸の長さは鳥種により差はあるものの、おおよそ **8〜28.5mm**。

| 動物 | 眼球の長軸長 |
|---|---|
| 小鳥〜家禽 | 8〜28.5mm |
| ヒト | 約24mm |
| ダチョウ（陸生動物で最大） | 50mm（3.5km先まで見える） |

眼球が大きいほど焦点距離が長くなり、解像度が高くなる。

---

## 3. レンズ機能（角膜と水晶体の両方）

哺乳類のレンズ機能は水晶体のみだが、**鳥類では角膜もレンズ機能を有している**。ハトの場合、角膜の湾曲は直径7.2〜9.2mmの幅で変わる。

### 3-1. 鳥類特有のレンズ調節機構

- **強膜骨**: 強膜内にある骨組織。水晶体・角膜の湾曲調節を強化
- **クランプトン氏筋**: 毛様体の筋。収縮で角膜の縁を内側に引き、角膜の凸度を上げる（近くを見る時）
- **ブリュック氏筋**: 側レンズを内側に押し、水晶体の中心へ圧力を伝え水晶体の厚みと湾曲を強くする
- **毛様体筋**: 哺乳類は平滑筋だが、**鳥類は横紋筋**。毛様筋の調節が任意に行われている可能性を示唆
- **側レンズ**: 鳥類で発達したレンズ調節器官。水晶体を支え、栄養補給する

### 3-2. 鳥眼球の3つの形状

| 型 | 該当する鳥 | 特徴 |
|---|---|---|
| 平状型 | 一般的な鳥 | 標準形状 |
| 球状型 | カラス類 | 角膜の湾曲・突出が中程度 |
| 管状型 | タカ・フクロウ | 角膜の湾曲・突出が強い |

猛禽に向かうにつれ角膜の突出が強くなる。

### 3-3. 紫外線の透過

ソウシチョウの眼球を用いた実験で、**水晶体は紫外線の320nmまでの波長を透過して網膜に届ける**ことが分かっている。**ヒトの場合、紫外線は網膜に到達することはない**。

---

## 4. 網膜櫛（pecten oculi）

ヒトや多くの哺乳類には網膜中心動脈があり、網膜の栄養を担っているが、**鳥には網膜中心動脈はない**。代わりに**網膜櫛（pecten oculi）**という鳥類特有の組織が網膜栄養供給を行う。

- 名前の通り櫛状の形をし、視神経の出口付近から硝子体に大きく突出
- 多数の毛細管が色素層から水晶体に向かって走行
- 内壁に多くの孔があり、構造的に物質の通過に関与
- 黒色色素を持ち、全体として黒色を呈する
- 機能: (1) 網膜への栄養供給、(2) 眼圧低下によるレンズ調節の補助、(3) 網膜上の影で外影の移動を認知しやすくする（Meyer 1986 提唱）
- **昼行性のタカ類で特に発達**

---

## 5. 視細胞と油球

### 5-1. 視細胞の2種

- **桿体（かんたい）**: ロドプシンを持ち、490〜506nmの波長を吸収。光感度は高いが応答・明順応は遅い
- **錐体（すいたい）**: 各種光波長の光吸収を行う視物質を持ち、色を識別する。感度は低いが応答・明順応は速い。単一錐体と複合錐体の2種類があり、どちらも外節（視物質が存在し光を受容する部位）との間に**油球を1個持つ**

### 5-2. 油球（oil droplet）

油球は錐体細胞内の細胞内小器官。新鮮な網膜を顕微鏡で観察すると、**赤色・黄色・緑色・透明**の綺麗な球として確認できる。研究者により分類は異なるが基本は4種類（黄色と緑色をさらに再分化して6種類とする論文もある）。

機能:
1. **フィルタの役目**: 一定範囲の光波長だけを吸収して他の成分をカットし、感光色素に到達する光の波長範囲を狭める
2. **集光機能**: 凸レンズと同様に光を集める

油球は中心部で小さく、網膜周辺部では同じ色の油球でも大きい。視覚精度が下がる周辺部で集光機能を強化するための仕組みと考えられる。

油球の色の構成は鳥種により異なる。

- **カラス**: 透明・青色・緑色・赤色の油球がほぼ同じ割合
- **カモ**: 黄色油球が優位

これは鳥種により得意とする波長や必要な色が異なることを示唆。食性の違いと餌の色の特殊性を見分ける戦略の違いと考えられる。

## 6. 視物質と色覚

### 6-1. ニワトリの視物質は5種類

**ニワトリは5つの視物質を持ち、そのうち4種類が色を見る視物質**である。一方ヒトは3種類のみ。

| 視物質 | 最大吸収波長 |
|---|---|
| ニワトリ ロドプシン | 503nm |
| ニワトリ赤 | 571nm |
| ニワトリ緑 | 508nm |
| ニワトリ青 | 455nm |
| ニワトリ紫 | 415nm |
| ヒト赤 | 660nm |
| ヒト緑 | 531nm |
| ヒト青 | 417nm |

**ヒトは光三原色（赤・青・黄）で色覚を構成するが、鳥類は光四原色で色覚を形成する。** ニワトリではヒトより1種類視物質が多く、色覚を形成する上で有利な条件を持っている。

### 6-2. 短波長領域の優位性

視物質の光波長吸収曲線を比較すると:

- **ヒト**: 緑波長と赤波長は比較的近く、青波長が離れた位置にある → 赤・緑付近の識別はよくできるが、短波長の分解能は低い
- **ニワトリ・ヨーロッパムクドリ**: 吸収スペクトルの高い位置が等間隔に並んでいる → 青波長や紫波長のような短い波長でも色弁別能が高い

### 6-3. 紫外線域の感受

七面鳥やヨーロッパムクドリでは、視物質の吸収波長は紫外線域にも反応することが明らかになっている。**ヨーロッパムクドリではニワトリより短波長の375nm にも高い吸収波長があり、紫外線も可視範囲と考えられる。**

> **ニワトリへの含意**: ニワトリのニワトリ紫（415nm）はヒトには見えにくい紫色領域だが、紫外線域までは伸びていない。ただし水晶体の透過特性は紫外線を通すため、視物質の感度域では一部の紫外線情報を受容している可能性がある。

## 7. 中心窩と側頭窩

網膜の解像度は、神経節細胞と視細胞の密度によって決まる。網膜神経節細胞密度の最も高いところを**中心窩**といい、最も視覚精度が高い部位。

**鳥類の多くは中心窩に相当する部位が2箇所ある** — 中心窩と**側頭窩**。

- **中心窩**: 片眼視において最も精度の高い場所
- **側頭窩**: 水晶体の屈折率から考えて両眼視野形成において重要な場所

これは鳥の眼球が側頭についており、前方への視覚精度を上げる必要から進化の中で獲得したものと考えられる。**鳥類は広い視野に加え、片眼視野・両眼視野ともに視覚精度の高いスポットを持つ。**

中心窩・側頭窩では視細胞から神経節細胞の情報連絡が最も集束も分散もしない部位であり、視細胞の数と神経細胞の数が近似する。中心窩では中継する細胞も少ないため網膜は薄くなり、その周辺は厚くなる。**この構造により網膜の他の部位とは異なり像は1.3倍も拡大される**。トビなどの猛禽類はこの効果により遠くの対象物をはっきり捉えることができる。

---

## 8. 神経節細胞数の比較

| 動物 | 神経節細胞数 | 視細胞密度 |
|---|---|---|
| ヒト | 約100万個 | 約1万個/mm² |
| カモ | 250万個 | — |
| カラス | 360万個 | — |
| キセキレイ | — | 12万個/mm² |

**鳥類はヒトより神経節細胞数が圧倒的に多く、視覚情報を取り入れるための機能が高次化している**ことが分かる。これらの神経細胞は網膜内で一様に分布しているのではなく、中心部で最も密度が高く、周辺になるにしたがい密度が低くなる。

---

## 9. 鳥類が哺乳類より優れた視覚を持つ仕組みのまとめ

| 要素 | 鳥類 | ヒト・哺乳類 |
|---|---|---|
| 眼球の大きさ | 脳と同じかそれ以上 | 脳より小さい |
| 視野 | 広い（200〜359°） | 200° |
| レンズ機能 | 角膜＋水晶体 | 水晶体のみ |
| 毛様体筋 | 横紋筋（任意調節可） | 平滑筋 |
| 強膜骨 | あり | なし |
| 網膜中心動脈 | なし（網膜櫛で代替） | あり |
| 視物質の種類 | 5種類（ニワトリ） | 3種類 |
| 油球 | あり（フィルタ＋集光） | なし |
| 紫外線受容 | あり | なし |
| 中心窩 | 2箇所（中心窩＋側頭窩） | 1箇所 |
| 神経節細胞数 | 数百万個（カラス360万） | 約100万 |
| 色覚 | 4原色 | 3原色 |
