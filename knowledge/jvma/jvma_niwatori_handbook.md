# 概要

日本獣医師会が発行する「動物福祉ハンドブック」のニワトリ編（2004年）。坂井利夫（坂井利夫家畜診療所）の執筆によるもので、家庭・愛玩飼育を含む小規模なニワトリ飼育を想定し、**分類・歴史・品種、解剖学的特徴、生理学的値、飼育環境、餌、繁殖、ヒナの管理、疾病診断、薬剤治療**について体系的にまとめている。鶏の基礎生態と疾病情報を一冊で扱う本システム内で唯一の包括資料。

## 1. 分類・歴史・品種

### 1-1. 分類

ニワトリの学名は *Gallus gallus*。動物学的分類は次の通り。

| 階層 | ラテン名 |
|---|---|
| 脊椎動物門 | Phylum Chordata |
| 脊椎動物亜門 | Subphylum Vertebrata |
| 鳥綱 | Class Aves |
| 新鳥亜綱 | Subclass Neornithes |
| キジ目（鶏形目） | Order Galliformes |
| キジ科 | Family Phasianidae |
| ニワトリ属 | Genus *Gallus* |
| ニワトリ種 | Species *gallus* |

### 1-2. 歴史

ニワトリ属（*Gallus*）には4種の野鶏が属する。

- 赤色野鶏 *Gallus gallus*
- 灰色野鶏 *Gallus sonneratii*
- セイロン野鶏 *Gallus lafayettii*
- アオエリ野鶏 *Gallus varius*

ニワトリの祖先（野生種）は、現在もインドから東南アジアにかけて野生する**赤色野鶏**であると考えられている。これは、進化論で有名なダーウィンが、これら4種の野鶏と現在のニワトリとの間に雑種ができるが、この一代雑種に繁殖力のあるのは赤色野鶏との雑種だけで、他の野鶏との間の一代雑種は一般的に繁殖力がないことから、赤色野鶏を祖先とする説が有力であるとした。しかし最近では他の野鶏との雑種にも繁殖力があることがわかり、この説は見直され、近年は遺伝子による解明が行われている。

主要な祖先である赤色野鶏は、インド・ビルマ・マレー・タイなどの地域で**約5000年前に家畜化**されたと考えられている。ニワトリの実用的な経済動物としての育種はローマ帝国時代に始まったが、近代的な卵用鶏や肉用鶏等の育種の歴史は比較的浅い。

### 1-3. 品種

ニワトリの品種には、現存する野生種と、家畜や愛玩用として改良された数多くの品種がある。

#### 卵用種

- **白色レグホーン（White Leghorn）**: イタリア原産。地中海沿岸種と呼ばれた代表的な卵用種で、日本含め世界中の農場で最も多く飼育されている。羽色は純白、単冠、皮膚色・脚色は黄色。卵殻は白く、就巣性はない軽量種。
- **褐色レグホーン（Brown Leghorn）**: レグホーンの品種で羽色は赤笹（赤色野鶏と同じ羽色）、白色卵を産む。イギリス含めヨーロッパで多く飼育、日本では愛玩用の飼育がわずか。
- **黒色ミノルカ（Black Minorca）**: 地中海のミノルカ島からイギリスに輸入され改良された卵用種。単冠またはバラ冠で羽色は黒、卵殻色は白、体重が大きいため現在では鑑賞用が中心。番性のミノルカも作出されている。

その他、南米チリのアラウカ地方の **アローカナ（Araucana）** は青または緑色の殻の厚い卵を産むことで有名。アメリカのカリフォルニアグレイ等もある。現代の養鶏場で飼養されている銘柄は、系統間交配や数種の品種を交配して作出した基礎鶏群を近親交配した近交系雑種（incrossbred）が多く、代表はシェーバー、デカルブ、ハイライン等。

#### 卵肉兼用種

- **横斑プリマスロック（Barred Plymouth Rock）**: 1874年アメリカで公認された卵肉兼用種。羽色は黒白横斑で伴性遺伝、雄はやや白っぽく雌はやや黒っぽい。卵殻は赤褐色、就巣性はほとんどない大型種。白色レグホーンとの交配種は卵用種のロックホーン。
- **ロードアイランドレッド（Rhode Island Red）**: 赤色マレーゲーム、レグホーン、アジア起源の在来種の交配からでき、1904年公認。羽色は赤褐色で尾羽・主翼羽・尾羽の先が黒いコロンビア型。卵殻は褐色。白色レグホーンとの一代雑種は卵用種のロードホーン。
- **ニューハンプシャー（New Hampshire）**: ロードアイランドレッドの改良種で1935年（昭和10年）公認。一時ブロイラーの作成用種として使われた。
- **オーストラロープ（Australorp）**: オーストラリアで黒色オーピントンを改良して作出。羽色は黒、単冠、卵殻は淡褐色。
- **名古屋種（Nagoya）**: 明治初期、各地で中国原産のバフコーチンに地鶏と外国種を交配して卵肉兼用の実用鶏を作出。名古屋種はこれらコーチン系実用鶏の代表的品種で明治38年（1905年）に日本家禽協会によって名古屋コーチンとして公認、大正8年に名古屋種と改称。昭和初期まで日本の主流。羽色は淡い褐色（バフ）で尾羽・主翼羽の先は黒色。単冠、肉垂は鉛色、卵殻は褐色、就巣性があることが多い。卵用の小型種（I型）と肉用の大型種（II型）がある。

#### 肉用種

- **白色コーニッシュ（White Cornish）**: イギリスで作出され、アメリカで日本の大シャモと交配・改良。ブロイラーの雄系として最も重要な肉用品種。
- **白色プリマスロック（White Plymouth Rock）**: 横斑プリマスロックの突然変異から改良。近年はブロイラー作成用雌として白色コーニッシュの雄とともに重要な肉用品種。
- **コーチン（Cochin）**: 中国原産。横から見ると正方形に近い独特な体型。コーチンはイギリスやアメリカで卵肉兼用種の育種に果たした役割は大きかったが、現在では観賞用に少量飼育。

#### 愛玩用種（日本鶏中心）

ペットとして飼育されるものや、闘鶏、その鳴き声を楽しむ品種が数多く見られる。

- **軍鶏（しゃも、Shamo）**: 江戸時代初期にシャム（タイ）から日本に入ったマレー系のニワトリと考えられる。「シャモ」の名もシャムがなまったとされる。闘鶏用である大型の大軍鶏、中型の中軍鶏と愛玩用の小軍鶏がある。冠は3枚冠あるいはくるみ冠で赤耳染、脚は黄色。胸の肉付きが良いので米国でブロイラーの雄の改良に使われた。日本でも地鶏と交配された。固定されて品種となったものが比内鶏、声良、薩摩鶏等。小型のものはバンタムと称する。
- **東天紅（とうてんこう、Totenko）**: 日本の長鳴鶏は東天紅、声良、唐丸の3種で、その中で最も早く昭和11年（1936年）に天然記念物に指定。江戸時代に作出されたといわれる高知県産の鶏種。鳴くことの長いことで有名で、普通の雄鶏の鳴き声は1〜2秒なのに対し25秒に達するものもある。単冠で耳染は白、脚は柳（緑）色。
- **尾長鶏（おながどり、Japanese Long-tailed Fowl）**: 高知県産の日本鶏の代表的品種で、12mに達する尾羽を持つ。大正12年（1923年）に天然記念物指定。羽色は白笹、赤笹、白の3種類。単冠で白耳染、脚は鉛（青）色。小国（しょうこく）から作成された。
- **小国（しょうこく、Shokoku）**: 平安時代に遣唐使によって中国から入ったと伝えられるが確証はない。羽色は白笹または五色（白笹に赤い羽が混じる）、単冠、赤耳染、脚は黄色。尾羽が長く、尾長鶏・東天紅とも近縁と考えられ、長鳴性と闘争性を兼ね備える。
- **土佐地鶏（Tosa Native Fowl）**: 明治維新以前から飼育されている地鶏を日本鶏と呼び、合計17品種が公認。土佐地鶏はそのうちの最古で小地鶏とも呼ばれる。高知県原産、羽色は赤色野鶏と同じ赤笹、赤耳染、脚は黄色。
- **岐阜地鶏（Gifu Native Fowl）**: 岐阜県原産の地鶏で普通地鶏とも呼ばれる。羽色は赤笹、単冠、赤耳染、脚は黄色。
- **尾曳（おびき、Long Saddle Bantam）**: 一名 芝曳千鶏ともいわれ、高知県の原産。小型で羽色は赤笹、単冠、耳染は白く、脚は柳（緑）色。
- **鶉尾（うずらお、Japanese Rumpless Bantam）**: 序鶉鶏とも呼ばれ、おそらく土佐地鶏の突然変異によってできたと思われる。尾骨を欠き無尾。羽色は赤笹と白色のものがある。単冠、白耳染、脚は黄色。
- **チャボ（Japanese Bantam）**: 原産地はベトナムの占城（チャンパ）とされ、「チャボ」の名もこれに由来するといわれる。江戸時代初期に中国から渡来し、江戸時代末期、明治、大正、昭和の4代にわたり育種改良された国際的な鑑賞・愛玩鶏。昭和16年に「チャボ」という名称で天然記念物指定。羽装（丸羽あるいは半丸羽の雌型羽装、絹糸、捻縮羽）、羽色および冠の大きさによって25の品種がある。体型は胸が前に強く張り出し、尾羽が大きく直立、短脚で大きい翼が地面に達することが主要な特徴。冠は肉垂とともに発達した大型と中型がある。関西以西では大型、関東では中型のものが好まれる。
- **烏骨鶏（うこっけい、Silkie）**: 原産地はインドで東アジアに広く分布。江戸時代に中国から日本に入り、昭和17年天然記念物指定の日本鶏。羽は白または黒で、羽性は糸毛（シルキー）で羽の小羽枝が絹糸状にバラバラになっている。骨、皮膚、肉垂に黒色色素がつく。毛冠を持ち、5趾、耳染は碧藍色。脚羽がある。

## 2. 解剖と生理（各部の構造と機能）

ニワトリは鳥綱として、爬虫類と哺乳類の中間的な体勢を持つといわれるが、一般的には爬虫類に近い特徴的な体勢を持つ。しかし哺乳類と類似する点も多々ある。これらの特徴はアヒル、ウズラ、クジャクにも当てはまる。

### 2-1. 体表（羽毛・皮膚）

体表は爬虫類の角鱗から進化したものとみられる羽毛におおわれる。**ニワトリの皮膚は薄く、血管・神経が乏しいため外傷による出血は少ない。汗腺はない。** 羽は皮膚の表皮層から発達した鳥類に特有の角質器であり、発育中に換羽しながら成長していき、成長後も夏から秋にかけて年1回換羽が行われる。このほか、嘴・けづめ・脚鱗に包まれた下腿部などが特徴的である。

### 2-2. 換羽

鳥類は繁殖期を過ぎると換羽をして新しい綺麗な羽になる習性を持つ。**換羽の引き金は、卵巣機能の低下によるエストロジェンの分泌不足**といわれる。卵巣機能の低下は、秋から冬にかけての日照時間の減少や、病気による体力の低下、餌の不足によることもある。また、孵化後成鳥になるまでに数回の換羽を繰り返して成長する。

### 2-3. 消化器（胃は3つ?）

ニワトリを解剖すると、腸より上部の3カ所にエサが見られる。

1. **嗉嚢（そのう）**: 一番上は食道壁の一部が拡大した袋状の場所。口の中に歯のないニワトリは、ここに急いで食べたエサを一時的に貯留する。胃ではない。嗉嚢内ではアミラーゼによりデンプンの一部を消化する。
2. **腺胃**: 胃液（ペプシンを含む）を分泌する胃。
3. **筋胃（砂嚢）**: 消化・分解を行う胃。蓄えた小石による機械的消化と、ペプシンによる蛋白質分解を行う。

盲腸は一対でよく発達している。小腸は長いが結直腸は短く、末端は**総排泄腔**と呼ばれ、ここには**尿管も開口する**。このため、ニワトリの糞は尿と混じって排泄されるので、糞中に排泄される白い尿酸と混じって白く見えることがある（別に茶色の盲腸便もみられる）。

### 2-4. 呼吸器・鳴管・気嚢

鳥類の喉頭は哺乳類とは異なり、喉頭蓋はなく、スリット状の喉頭口が閉じて食物が気管に入るのを防いでいる。喉頭口より気管に連続し、さらに気管支に分かれて肺に入り、二次気管支、air capillary（肺のガス交換部）と続く。**横隔膜は痕跡的**である。なお、ニワトリの気管・気管支は軟骨輪が完全に閉鎖し、哺乳類のような背部の膜性部はない。

- **鳴管**: ニワトリには哺乳類の喉頭にある声帯はない。気管が胸部で気管支に分かれて肺内の二次・三次気管支へ続く。気管支胸部が偏平な形をした部分があり、これが鳴管である。
- **気嚢**: 鳥類に特有の器官で、気管支枝の一部が肺を貫通し、さらに肺外の内臓間、筋肉間、骨質中に延び拡大したもの。気嚢は薄く透明な2層の膜でできており、これら2層間に少量の結合織を挟む。内層は気管支から連続の粘膜で、外層が体腔漿膜の連続。気嚢の働きとしては、(1) 鳥の体を軽くし、(2) 呼吸を効果的に行い、(3) 体内で発生した熱を冷却することが考えられる。

### 2-5. 泌尿器

ニワトリの泌尿器は左右一対の腎臓と尿管からなり、**膀胱・尿道はない**。そのため腎臓より濾過された尿は尿管を通り、総排泄腔に入り糞と一緒に排泄される。哺乳類は体内の窒素を尿素の形で水中に排出するが、鳥類は主に**尿酸**の形で排出する。このため、腎臓障害が生じたとき容易に**尿酸塩沈着症（痛風）**が起こるといわれている。

### 2-6. 生殖器

すべての鳥類の雌は胚の時には一時期、右の卵巣と卵管が存在するが、やがて退縮して、孵化した時点でタカ類以外の鳥類で**左の卵巣・卵管だけ**が見られる。また、右側の卵管はミューラー管として遺残し、時に嚢腫をつくる。雄の精巣は腹腔内の腎臓の前部、腹側に一対あり、精液は精路を通じて総排泄腔に送る。

### 2-7. 卵のできかた

卵巣は背中の中程（腰仙骨部）にあるが、成熟期になると肝臓から送り込まれる卵黄物質によりブドウの房状に発達し、やがて成熟した卵胞の中の卵黄が卵管のロート部と呼ばれる部分に入って行き（**排卵**）、さらに卵管膨大部でカラザや卵白が分泌される。次に卵管峡部で卵殻膜、子宮部で20時間程かけて回転しながら炭酸カルシウムの卵殻が形成され、膣部を経て産卵される。**おおよそ25時間ほどのサイクル**である。

### 2-8. 骨格

ニワトリの**頚椎は14個で哺乳類の倍**である。また鳥類に特有な構造として、気嚢により骨が**含気骨化**して、軽く丈夫な構造を持つ。また、産卵している個体は骨腔の中に卵殻を形成するための特別なカルシウムの貯蔵所として**髄骨**を持つ。ここでは卵殻の形成状況によって破骨細胞や骨芽細胞がカルシウムの収支バランスを取っている。

### 2-9. ファブリキウス嚢（F嚢）とハーダー腺

**ファブリキウス嚢（F嚢）**は鳥類に特有のリンパ上皮性の器官で、総排泄腔の背側にある。F嚢はBリンパ球を産生する中枢リンパ器官で、内腔は表層上皮で覆われる。リンパろ胞が内側に接する場所では抗原などをろ胞内に取り込む機能があるといわれている。ニワトリでは20〜27週齢までには完全に退縮する。

**ハーダー腺**は眼窩後方の外側にあり、瞬膜の潤滑性を保つための分泌腺だが、固有層にはリンパ組織が見られる。2週齢以降発達して成鳥でも認められる。

鳥類には哺乳類のようにリンパ節がないため、ほかに免疫にかかわる器官として盲腸扁桃、脾臓、骨髄など、気管や腸の粘膜固有層のリンパろ胞様のリンパ球集合が働いている。

### 2-10. 循環器と血液

鳥類の心臓は二心房二心室であり、**成熟した赤血球でも有核**である（哺乳類の赤血球は無核）。

### 2-11. 肉冠・肉垂・耳染

これらはニワトリ類特有のもので、皮膚の真皮が肥厚し、この部分は動静脈吻合に富む毛細血管網で形成される。肉冠には単冠、バターカップ、V型冠、ばら冠などがある。耳に耳介はなく、耳小骨は小柱骨一つである。白色レグホーン種のような耳染の白いものは、真皮の毛細血管網がほとんどなくなり、**白色の色素プリンが沈着したもの**。

> **編集者注**: 原本は「色素プリン」とのみ表記。動物界で代表的な白色色素のプリン塩基は**グアニン**であり、本記述もグアニンを指している可能性が高い（魚のうろこの銀色や鳥の羽の白色を作る既知の生物色素）。

## 4. 飼育環境

### 4-1. 飼育舎

ニワトリを飼育する場合、一般的には屋外の飼育舎でコンクリートや土間の床が多い。広い場所で自由に運動させ、生態を観察するには一番良い方法。しかし、野外飼育は常に目が届くわけではないので、事故や病気の発生しにくい構造がよい。

### 4-2. 気温

**ニワトリは暑さに弱く寒さに強い。** 入梅明け前後のその年一番の暑さの日は熱射病の危険日である。冬に冷たい強い風が吹くと頭を風上に向け、体を低くする — **強い風が嫌いなようである**。夏は西日を避け、冬は反対に日当たりが良く冷たい風が吹き込まない構造と場所が確保できるとよい。トタン板の屋根の下には断熱材が必要である。

### 4-3. 床と運動場

基本的に多湿の状態が苦手である。運動場の砂や土は掘り起こして常に柔らかくし、**砂浴び**ができると外部寄生虫の対策になる。放し飼いのニワトリの糞は棲架（止まり木）の下と餌の周囲に半分以上集中するので、この場所を掃除するとよい。

気温が高くなると、汗腺のないニワトリは呼吸数と飲水量を増やして体温を下げようとする。夏期の飲水量は冬期の3倍にもなるので、糞は水っぽくなり、床も湿りやすい。掃除の時に糞の色や形状を観察して健康状態を把握する（緑便、血便、下痢便…）。

### 4-4. 棲架（止まり木）

ニワトリが夜間に寝る場所で、飼育舎の一番奥の安心できる所に作る。**壁から30cm離し、床から1mくらい**がよい。**一羽に30cmの幅**を用意する。横木に殺虫剤や防腐剤を塗っておくとダニなどの外部寄生虫対策になる。

### 4-5. 餌箱と水入れ

ニワトリは配合飼料のような粉状の餌を食べる時は水と交互に口に運ぶので、餌箱と水入れは並べて置くのがよい。餌箱の上に乗ってかき散らす性質があり、餌箱や水入れに脚が入らないように棚を設け、高さは胸（嗉嚢）の位置がよい。また、餌箱と水入れの下や付近は湿り易く、糞の量も多いので不潔になりがち。この場所は一段高くして、すのこやケージにすると衛生的であり、糞の処理も楽である。

### 4-6. ケージ（隔離が必要な場合）

ニワトリは**群の生活に慣れている**が、新入りに対してはいじめる時がある。また、雄同士は縄張りをめぐる闘争が起こるので隔離が必要になる。この場合はケージに収容するが、すのこは脚が落ちない・糞の詰まらないくらいの細かさが望ましい。餌入れは壁掛け式がひっくり返されず清潔である。

### 4-7. 産卵箱

産卵している羽数の **3羽に1個の割合**でよい。**薄暗くしておく**。あまり明るいと他のニワトリに肛門をつつかれて脱肛や腸の損傷を受けて死亡するものもあり、注意が必要である。

## 5. 餌

### 5-1. 成鳥

ニワトリは**雑食性**であるから、穀類、種子類、動物性蛋白質、緑餌（葉野菜など）、ミネラルなどが餌となる。ただしバランスが重要で、**カルシウムの不足でくる病が発生**したり、**カロリー過多により脂肪肝**の症例がある。

#### 配合飼料

最近では配合飼料の利用が多い。粉砕した穀類、魚粉、乾燥した植物、カルシウムやリンなど微量成分を配合したもの。穀類は粉砕していないと消化率が低い。産卵している個体がいて卵をたくさん取りたければ産卵鶏用を用いる。野菜・果物が手に入る時は併用して与える。

#### 野菜・果物

人の食べているものは基本的に与えてよい。原本では「ネギ、ニラ、ニンニク等も大量でなければ元気のでる食品である」と記載されている。

> ⚠️ **編集者注**: 原本（2004年）の上記記述については、その後のペット栄養学では一般にネギ属（ネギ・ニラ・ニンニク・タマネギ）はイヌ・ネコで赤血球溶血を起こすとされ、鶏に対する安全性も研究によっては毒性が報告されている。**鶏に対する安全性は専門家への確認が望ましい**。本ハンドブックの記述をそのまま採用する前に、現代の獣医学的合意を確認すること。

### 5-2. 育成期

ニワトリは生後12週齢までに、**骨格の大きさ（フレームサイズ）がほぼ決まってしまう**といわれる。また、ビタミンやミネラルの不足による疾病や発育異常が数多く知られている。これらのことから配合飼料の利用が安心である。育成用の配合飼料は産卵鶏用では:

- **幼雛用**: 生後4〜5週齢まで
- **中雛用**: 4〜5週齢以降10週齢まで
- **大雛用**: 10週齢以降から産卵を始める頃まで

の3種類があり使い分ける。

## 6. 繁殖

孵化には母鶏を用いる場合と、孵卵器を用いる場合がある。**孵卵期間は21日間**だが、様々な条件を満たさなければ孵化率は良くない。

### 6-1. 種卵

交配のための雄と雌の比率は、雄1羽に雌5〜10羽といわれているが、雌4〜5羽がよい。卵の大きさはあまり大きかったり小さいものは避け、中くらいのものを選ぶ。卵の保存温度は **10〜16℃**、湿度は **75%前後** が最適。保存期間は7日以内では孵化率に影響しないが、21日間では50%近くまで低下するといわれる。また、転卵しながら保存すると孵化率の低下が軽減したという報告がある。

### 6-2. 母鶏孵化

卵を温める巣鶏が必要。**就巣性の残る、改良の進んでいない日本鶏やチャボの雌が選ばれる**。事前に介卵性の病気を治療したり、ダニなどの寄生虫の駆除をしておく。巣箱は薄暗い静かな所に置いて、底に3cmくらいの厚さに湿った土を敷きその上に藁を入れ皿状にする。母鶏は1日に1〜2回採食と排泄のために巣から出る。これは孵卵器でも必要だが、卵の周囲の温度変化や酸素の供給が卵の発育に良い効果を与える。母鶏用の餌や水と砂浴び施設を近くに設けておく。

> 介卵性の病気: 鳥類では母鶏の病気（や抗体）が卵を介してヒナに伝わる。時には孵化の途中で発病して死んだり、虚弱なヒナが生まれる（病気の予防が重要）。

### 6-3. 人工孵化

電気孵卵器による孵化で、温度は **卵の上端で38.9〜39.4℃** に保つ。湿度は **40〜70%** が最適。孵卵中の卵は**気室のある鈍端を上**にする。また、卵を回転する**転卵**も重要な作業。回数は多い方が成績が良く、1日3回転卵した時の孵化率は **74.7%**、1日24回転卵の場合の孵化率は **78.0%** という報告もある。転卵の角度は **45〜60度** がよい。

### 6-4. 検卵

孵卵中に検卵を行って無精卵や発育中止卵を取り除いたり、胚の発育状態をチェックする。検卵は上部に1.5cmくらいの穴をあけた箱の中に電球をつけ、上に卵を置いて暗い部屋の中で観察する。3回行うが、1〜2回で判断する人もある。検卵の日とポイントは次のとおり。

- **1回目（孵卵5〜7日目）**: 発育卵はやや赤色を呈し、胚から四方に血管が伸びている。
- **2回目（孵卵12〜14日目）**: 発育中のものでは気室が明るく、それ以外は暗いので気室との境界が明瞭。気室の近くまで血管が走っている。胚が時々動く。
- **3回目（孵卵18日目）**: 発育卵は全体に黒く見え、気室の容積が拡大してくる。気室と胚の境界のところで胚が動いているのが観察される。

## 7. ヒナの管理

### 7-1. 温度と湿度

孵化したてのヒナの数が少ないと温度は保ちにくいが、**4〜5週間は保温が必要**。育雛器の温度は、初めは **36〜37℃** に設定し、2〜3日はその温度で換気と湿度に注意しながら様子をみる。以後 1週間に2〜3℃ずつ下げて4〜5週間で**廃温**（保温をやめる）する。この時の室温は18〜20℃程度がよい。

育雛器の湿度は初めの3〜4日は **70〜75%** が必要。育雛器の内側にタオルを下げ、下の水盤の水に浸けるようにして湿度をとる。湿度は10日齢以降は少しずつ下げて **40〜60%** 程度でよい。照明は初めの3〜4日は24時間で、以後少しずつ減らして自然日長にしていく。

### 7-2. 雛の餌付け

孵化してから **25〜60時間** くらいして、ヒナに初めての餌を与える。これを **餌付け** という。ヒナがこの間なにも食べないのは、卵の時代の栄養分が体内に残っているから。初めての餌は配合飼料の幼雛用がよいが、手に入らない時は水に浸した小米、または小鳥の餌（アワやヒエの入った）を煮たものとゆで卵の卵黄を砕いたものを混ぜて与える。

初め餌を食べないようであったら、紙の上でヒナの近くにバラバラと餌を撒いてみる。また、餌のある所を尖らせた割り箸でつついてみると、ヒナはその動きを追ってマネをしながら餌付く。

ヒナは軽くつかんで水入れに顔を入れ嘴を濡らしてやるか、水面がよく見える位置にして（目の高さが良い）やると興味をもって突ついて水を飲むようになる。

餌は成長するに従って魚粉や野菜など与えてゆくか、配合飼料に変えてゆく。

## 8. 病気の発見と現場対応

> 本セクション以降は疾病・治療情報を扱う。**実際の診断・治療は必ず獣医師に相談すること。** 家畜伝染病予防法における法定伝染病・届出伝染病の発生疑いがある場合は、家畜保健衛生所へ連絡する。

### 8-1. 病気の発見

病気の発見には臨床症状と各種の病気との関連を頭において診療を行う。生きている場合には採血を行って各種感染症の抗体検査を依頼し、また糞便や血液塗抹標本などにより寄生虫検査を行う。死亡している場合には剖検から病理検査が行えるとよい。

### 8-2. 採血と糞便検査

採血は翼下にある **尺側皮静脈** からゆっくりと行う。

1. 翼の内側をアルコール綿で消毒し、目的の静脈がよく見えるように不要な羽毛を取り除く。
2. 長橈側手根伸筋の下から上腕部尺側皮静脈に採血針を入れ、ゆっくりと採血する。
3. 全血のままと血清での検査が行われる。

#### 検査の用途

- **全血**: 市販のヒナ白血急速診断用菌液と等量をガラス板の上で揺振し、凝集塊の出現時間で感染を判定する。ロイコチトゾーン症の診断ではスライドグラスに末梢血液塗抹標本を作成し、ギムザ染色後メロゾイトまたはガメトサイト原虫を鏡検する。なお、この場合は肉冠を針で刺して血液を採ることもある。マイコプラズマ症の感染の有無にも全血または血清にて急速凝集反応用菌液が市販されている（マイコプラズマ・ガリセプチカム MG とマイコプラズマ・シノビエ MS の2種類）。
- **血清**: 血清を使って検査できる疾病としてはニューカッスル病や伝染性コリーザがあり、抗体測定用の抗原が市販されている。またロイコチトゾーン症や**鶏脳脊髄炎（Avian Encephalomyelitis, AE）**のゲル内沈降反応用抗原も市販されている。

正常な糞には **直腸便（黄褐色の部分）** と **盲腸便（チョコレート色の部分）** がある。

### 8-3. 現場での対応

診断依頼の連絡があった時、現地に到着するまでに次のような手順を依頼しておくとよい。

1. **病鳥はすぐに隔離してもらう**（体の大きさにあった段ボール箱などに紙を敷いて）。これは、病鳥が他の鳥に踏みつけられないためであり、糞をしていれば検査用に採取しやすい。
2. **病鳥は寒い時期のときは保温してもらう**。
3. **死亡している場合は冷蔵してもらう**（剖検・病理検査用）。

### 8-4. ワクチン接種の一般原則

ニワトリのワクチンの接種や薬剤の注射は、**頸部皮下接種** と **大腿部筋肉内接種** が多い。具体的なワクチンの種類・投与量・スケジュールは獣医師の指示および家畜伝染病予防法等の公的マニュアルに従うこと。

### 8-5. 補足: 鳥類の呼吸器感染症が重篤化しやすい理由

鳥類のガス交換は哺乳類の肺胞ではなく、肺の **air capillary** で行われる。**鳥類の空気血液関門は哺乳類より幅が著しく薄く**（ニワトリ約 0.30μm、アカゲザル約 0.65μm）、呼吸器に感染した細菌が血流に侵入し細菌血症になりやすく、呼吸器疾病が重症化する一因となっているといわれている。

### 9-3. ワクチン・薬剤の接種部位

ニワトリのワクチンの接種や薬剤の注射は、**頸部皮下接種** と **大腿部筋肉内接種** が多い。

## 10. 呼吸器疾病

### 10-1. ニューカッスル病（ND）

- **発生**: 強毒ウイルス感染による伝染病。最近ではコジュケイやインコ類、ハトの発生も報告。
- **原因**: パラミクソウイルス科の **NDウイルス** による。ほとんど病原性を示さない弱毒株から、90%を超える致死率を引き起こす強毒株まで多様。
- **症状**: ウイルス株の違いによりアジア型とアメリカ型の2つの病型に大別される。
  - **アジア型**（急性致死型・内臓型）: 食欲廃絶、嗜眠、濃緑色下痢、開口呼吸などの呼吸器症状を呈し、発症後1〜3日で急死。死亡率は100%に達することもある。経過が長引いた場合に首曲りなどの神経症状。
  - **アメリカ型**: 緑色下痢、呼吸器症状および神経症状を発現し、経過が長い。重症例では2〜7日で死亡。死亡率は成鳥で5%程度、ヒナでは50%。回復例では神経症状を呈する。
- **治療**: **淘汰することが基本**。家畜伝染病予防法では**法定伝染病**で、**届出義務**がある。ワクチン接種による予防が重要。

### 10-2. 伝染性喉頭気管炎（ILT）

- **発生**: ウイルスによる急性呼吸器病で、家畜伝染病予防法で**届出伝染病**に指定。窒息死もある疾病。新しいニワトリの導入が発生の引き金になることがある。
- **原因**: ヘルペス群に属する **伝染性喉頭気管炎ウイルス** の感染による。
- **症状**: 流涙、眼の周囲の腫脹、鼻汁漏出、発咳、喘鳴、喀血などの呼吸器症状。窒息死することもある。症状の軽いタイプもあり様々。
- **診断**: 鶏痘との鑑別が重要。組織検査では核内封入体の確認を行う。
- **治療**: 生ワクチンがある（点眼接種）。

### 10-3. 伝染性コリーザ

- **発生**: 細菌によって起こる急性の呼吸器病。他の細菌（マイコプラズマ、大腸菌など）と混合感染して慢性化する。
- **原因**: ***Haemophilus paragallinarum*（HPG）** の感染による。日本ではA型菌とC型菌がある。
- **症状**: 食欲の減退・廃絶、鼻汁排出、顔面ならびに**肉垂の浮腫性腫脹**、流涙、開口呼吸、下痢、緑便。産卵は停止し、時に卵墜症を起こす。
- **診断**: 細菌学的診断、抗体検査による。
- **治療**: ニューキノロン系およびマクロライド系（マイコプラズマにも効果のある薬剤）の併用が有効。

### 10-4. 伝染性気管支炎（IB）

- **発生**: ウイルスによる急性呼吸器病で、家畜伝染病予防法で**届出伝染病**に指定。感染鶏は呼吸障害、産卵障害、腎障害を示す。**ヒナ期感染**は成長後の産卵不能、**産卵期感染**は産卵率低下と卵質悪化。腎障害型は若齢鶏の死亡率増加。
- **原因**: 伝染性気管支炎ウイルスはコロナウイルスグループに属し、強い伝播力。ほとんどのニワトリが抗体を保有。
- **症状**: ゼーゼー、キャッなど激しい呼吸器症状と黄白色軟便・淡い緑便。鼻汁・眼の腫れはない。産卵停止または殻の無い卵を産む。回復期に奇形卵を産む、または完全に産卵停止する個体もある。腎炎タイプでは呼吸器症状を示さずぼんやりして、その後削痩・死亡する例。
- **診断**: ウイルス分離、抗体検査による。
- **治療**: ビタミン剤や抗生物質で体力維持・二次感染予防。

### 10-5. 熱中症（熱射病・日射病）

- **発生**: ニワトリは体温が **45〜47℃以上** になると死亡する。環境温度が急上昇または高温が長時間持続した場合、汗腺のないニワトリは呼吸回数の増加（**パンティング**）で熱を体外に排出しようとするが、この際に炭酸ガスを過剰排出し、**呼吸性アルカローシス**が死因となる。
- **原因**:
  - **熱射病**: 皮膚血管拡張により皮膚に血液が集中し、心機亢進・血圧低下などの循環障害でチアノーゼを経て虚脱状態。
  - **日射病**: 強い日射を受けて突然の高温で体温調整中枢が適応不能となり発熱。**入梅明けに多い**。
- **症状**: 口を開け、熱性呼吸症状、羽を広げて体温を下げようとする。夜間に肺炎で死亡する個体もいる。
- **診断**: 解剖して他の疾病と区別。
- **治療**: 首から下を水に漬ける。予防を兼ねて飲水または強制的に**炭酸水素ナトリウム**を投与（参考: ブロイラーの飲水に0.63%）。

### 10-6. その他の呼吸器疾病

カビ性肺炎（ヒナ）、インフルエンザ（家禽ペスト、§15-5参照）、マイコプラズマ感染症、鶏痘の粘膜型などが呼吸器症状を呈する。

## 11. 消化器疾病

消化器疾患は細菌や内部寄生虫の感染による急性発症が多い。発育期に多発し、重症例も多く、急死または発育不良に至る重要な疾病群。

### 11-1. コクシジウム感染症

- **発生**: ***Eimeria*** 属原虫の感染による。ニワトリに感染するのは **8種**。種により症状・寄生部位・病変が異なる。混合感染も多く、ニワトリ疾病の中で被害が大きいもののひとつ。
- **原因**: 胞子形成オーシストを経口摂取することで小腸や盲腸の上皮に感染し、有性・無性生殖により増殖。寄生部位・病原の強弱により3型に分類。
  - **急性盲腸コクシジウム症**: *E. tenella*（盲腸感染）
  - **急性小腸コクシジウム症**: *E. necatrix*
  - **慢性小腸コクシジウム症**: *E. acervulina*, *E. maxima*, *E. brunetti*, *E. mitis*, *E. praecox*, *E. hagani* の6種
- **症状**:
  - **急性盲腸**: 4〜5日の潜伏期後、血便、貧血、食欲不振・廃絶、仮眠後に死亡。ヒナの死亡率80〜90%に及ぶことも。剖検では盲腸のみに変化（粘膜出血・壊死、内腔の血液塊・チーズ様物充満、回復期は萎縮硬化）。
  - **急性小腸**: 4〜5日の潜伏期後、大量の粘血便を排出して死亡。剖検では小腸中部の点状出血、灰白色壊死巣、腫大、肥厚脆弱。
  - **慢性小腸**: 水様下痢便、粘液便、肉様便、体重減少、立毛が主症状。死亡は少ない。剖検では小腸の微細出血、点状/破線状壊死。
  - **共通**: 出血のあったニワトリの肉冠は貧血で萎縮。羽毛の色艶悪化、採食せずに一カ所にうずくまる。
- **診断**: 糞便の直接法または浮遊法検査でオーシスト確認。急性期はオーシストが見られないことも。死亡例は解剖し病変部粘膜を生理食塩水で希釈後鏡検し、シゾント・メロゾイト・オーシストを確認。
- **治療**: サルファ剤やST合剤の投与。**サルファ剤は1回投与3〜4日間とし、再投与時は3日間休薬**（サルファ剤中毒予防）。ウェルシュ菌との混合感染では合成ペニシリン同時投与が必要。

### 11-2. 壊死性腸炎

- **発生**: **ウェルシュ菌（*Clostridium perfringens*）** の小腸内異常増殖による。**温度・湿度が高い4〜9月に多発**。一度発生した場所は再発する傾向。
- **原因**: 原因菌の多くは**レシチナーゼを産生するA型菌**、ときにC型菌も検出。コクシジウム感染ニワトリの腸内ではウェルシュ菌の増殖が著しく、コクシジウム症重篤化に重要な役割。
- **症状**: 経過は非常に急激で、症状を現さずに急死することが多い。発病ヒナは元気消失、羽毛屹立、跛行。食欲なく、食べた餌を口からもらすことも。血液混じりの暗色便を排泄するが下痢はない。死亡直前に水を吐くことがある。発症ヒナは数時間で死亡。成鳥は慢性経過、削痩・貧血。
- **診断**: 小腸内容の塗抹標本をグラムまたはギムザ染色し、鏡検で大桿菌を確認。または指定培地による定量培養で菌数確認。
- **治療**: 早期にペニシリン系またはテトラサイクリン系などの抗菌剤の飲水投与が有効。

### 11-3. 内部寄生虫病

- **発生**: 国内のニワトリの内部寄生虫はほとんどが線虫。一部に吸虫・条虫・鉤頭虫。慢性経過で特徴的所見が少なく気づかれにくい。**鶏盲腸虫の虫卵は黒頭病の病原体ヒストモナス（*Histomonas meleagridis*）を媒介する**。
- **原因**:
  - **線虫**: 鶏回虫、鶏毛体虫（毛細線虫）、鶏盲腸虫、気管開嘴虫、鶏糞線虫
  - **吸虫**: 鶏盲腸吸虫、鶏卵吸虫
  - **条虫**: 有輪条虫、方形条虫、棘溝条虫
  - **鉤頭虫**: その他
  - 土壌と接する飼育形態が発生原因になりやすい。
- **症状**: 少数寄生・慢性経過では無症状、排泄された虫体で気づく。多数寄生では下痢・血便、削痩、羽毛汚れ。**気管開嘴虫は気管腔に寄生し呼吸困難・窒息死を起こすこともある**（キジの発生例あり）。
- **診断**: 糞便検査による虫卵検査、または排泄された片節の虫体検査。回虫卵と鶏盲腸虫卵の鑑別困難な場合多い。鶏糞線虫卵は体外排泄後短時間で孵化するため他種土壌線虫との鑑別必要。死亡例は気管・消化管を切開して虫体検索。条虫はストロビラが特徴的、鉤頭虫は米粒状。
- **治療**: 飼育舎の床材交換または消毒、**イベルメクチンなどの駆虫薬を投与**。

### 11-4. 黒頭病（原虫性盲腸肝炎）

- **発生**: ヒストモナス・メレアグリディス（*Histomonas meleagridis*）原虫の寄生による。**七面鳥への寄生が極めて多く激しい症状**を呈し、末期に肉冠が暗紫色になることから黒頭病と命名。**ニワトリの場合は慢性コクシジウム症のような経過で、鶏冠は黒くならず貧血・萎縮**。日本のニワトリでの発生は極めて少ないが、**七面鳥・アヒル等と混飼している場合は発生しやすく根絶困難**。消化器だけでなく**肝の疾患でもある**。
- **原因**: 新鮮な糞中の*H. meleagridis*の直接経口摂取は成鳥では胃酸で殺虫されるが、幼弱なヒナでは感染することがある。一般的な感染は、**盲腸虫卵の中に*H. meleagridis*が入っており、この盲腸虫卵を直接経口摂取するか、ミミズ体内の盲腸虫卵をミミズごと啄食して感染**。ミミズ体内の盲腸虫は土壌中のものより感染性が強い。
- **症状**: 感染3〜4日の潜伏期後、元気消失・食欲不振、黄色〜壁土色の下痢便排出、顔面の貧血、翼羽の下垂、肛門部の羽毛汚れ。**便は乾燥すると鮮やかなクリーム色**になる。ヒナの死亡率25%になることも。
- **診断**: 盲腸内原虫の鏡検は原虫が小型で無色透明・染色性不良のため無効。剖検で盲腸の病変（拡大、内部は組織壊死した黄白色チーズ様物充満・破裂）と肝臓の病変（**肝炎による腫大・硬化、黄〜暗緑色の菊花状の壊死病巣形成**）を確認、盲腸・肝臓の直接塗抹鏡検・病理組織検査で原虫を確認。
- **治療**: 飼育舎の土壌消毒・日光消毒。盲腸虫の駆虫（フェノチアジン）、ヒストモナス対策として**ナイサイアザイドを強制経口投与**（0.1〜0.2%を4〜6日連用）。

### 11-5. 食滞

- **発生**: 餌の不足、または笹などの根・羽毛・ビニールなどを飲み込んで**嗉嚢を起点に消化管を閉塞**することによる。
- **症状**: 食欲不振、元気喪失、削痩。時に口から羽毛や異物の一部を覗かせる。外観から嗉嚢部分の膨隆が認められ、原因物質によっては硬固感や水分貯留により波動感。
- **診断**: 触診で嗉嚢内容物の確認、腐臭などで判断。
- **治療**: 異物を口腔から、または外科的に摘出。

### 11-6. その他の消化器疾病

ヒナ白痢（§15-1参照）、鶏パラチフス、ボツリヌス症、カンピロバクター感染症、大腸菌症など。

## 12. 皮膚疾病

### 12-1. ブドウ球菌症（浮腫性皮膚炎）

- **発生**: 黄色ブドウ球菌（*Staphylococcus aureus*）の感染による疾病で、**浮腫性皮膚炎**が代表的。関節炎、趾瘤症、趾底炎、化膿性骨髄炎、初生ヒナの臍帯炎など多彩な病型。
- **原因**: *S. aureus* は皮膚・粘膜の外傷部位、気道などに感染するが、発病には誘因が必要。本菌は**溶血毒のアルファ毒素**（皮膚壊死毒および致死毒）、蛋白分解酵素、核酸分解酵素などを産生する。
- **症状**: 翼・腹・頸・胸・脚・背部の皮下に**茶褐色または帯赤色の漿液性滲出液**、皮膚は糜爛し異臭。一般に病変は翼の内側や先端部に多い。内臓型・敗血症の状態は沈鬱・食欲廃絶・下痢を示し1〜2日で死亡。肉冠・肉垂はチアノーゼまたは逆に貧血。肺の病変で死亡した例は口から帯赤色の液をみる。
- **診断**: 特徴的な皮膚病変による。細菌学的検査による。
- **治療**: 発症例の治療効果は低い。**予防的にペニシリン系・テトラサイクリン系・マクロライド系の抗菌剤投与**と発生要因（皮膚の損傷、免疫抑制疾病）への対応が重要。

### 12-2. 壊疽性皮膚炎

- **発生**: 主に**クロストリジウム・セプチカム**（*Clostridium septicum*）、場合によって**ウェルシュ菌**（*C. perfringens*）の感染で皮膚病変を伴って急死する疾病。
- **原因**: 皮膚傷害部位が嫌気的条件になり、クロストリジウムの侵入・増殖・発症が成立。多くの症例で**ブドウ球菌の混合感染**がみられる。
- **症状**: 食欲不振、沈鬱、脚弱、運動失調を示し、翼・胸・腹・脚部の皮膚が暗紫色を呈し腫脹・湿潤。**発症後の経過は短く、一般に24時間以内に死亡**。死亡例の肝は充血腫脹し、白色壊死巣がみられる。
- **診断**: 細菌学的検査。
- **治療**: ブドウ球菌症および壊死性腸炎の場合と同じ。

### 12-3. 鶏痘（皮膚型）

- **発生**: 急性のウイルス性疾患で、皮膚や粘膜に特徴的な発痘。粘膜型もあるが、ここでは皮膚型を扱う。
- **原因**: ポックスウイルス群に属する**鶏痘ウイルス**。皮膚型は**蚊やヌカカの吸血時に感染**が成立、夏から秋にかけての発生が多い。皮膚・粘膜の傷に直接ウイルスが付着しても感染。
- **症状**: 発痘の好発部位は無羽部、特に**肉冠・肉垂・眼瞼・口角**など。潜伏期間5〜6日、初めに灰白色の小丘疹が現れ次第に大きくなり黄色化し、中心部に出血。次いで全部が暗赤色の糜爛になる。数日後に乾燥した茶褐色の痂皮ができ、10日くらいで自然に脱落。皮膚型は粘膜型に比して回復しやすい。混合感染があれば経過は長くなり、粘膜型併発時は窒息死もある。
- **診断**: 発痘は特徴的で診断は容易。確定診断は組織検査（封入体の検出）かウイルス分離による。
- **治療**: 患部の**ヨードホール**などによる消毒、二次感染予防のためビタミン剤・抗生物質を投与。

### 12-4. 外部寄生虫病

#### トリサシダニ・ワクモによる皮膚疾病

- **発生**:
  - **トリサシダニ**: 秋に増殖、冬の寄生が最も多く夏に減少。主に**尾羽のつけ根、腹部、肛門周囲などの羽毛に寄生し、一生をニワトリの体表で過ごす**。現在22種の鳥類に寄生。人・齧歯類にも寄生する。
  - **ワクモ**: 夜間の吸血時のみニワトリを襲い、**日中は糞塊や建物の隙間などに潜む**（トリサシダニとの相違点）。**春から秋に増殖、冬は不活発**。
- **原因**:
  - トリサシダニ（*Ornithonyssus sylviarum*）: 雌成虫の体長 0.6〜0.7mm、体色は淡白〜深赤色
  - ワクモ（*Dermanyssus gallinae*）: 雌成虫の体長 0.8〜0.9mm、体色は赤〜黒色
- **症状**: トリサシダニは肛門周囲などの羽毛（皮膚から2cmくらいの位置）に寄生し、吸血と不快感でニワトリは神経質になる。**肛門周囲の皮膚にかき傷・出血・痂皮形成・貧血・産卵低下**。ワクモも吸血により同様な症状。
- **診断**: 虫体を50〜60℃の温水で殺し、1%エーテル加70%アルコールに入れ液浸標本として鏡検。
- **治療**: 駆除剤として**有機リン剤、カーバメート剤、ピレスロイド剤等**を投与。

#### ハジラミによる皮膚疾病

- **発生**: ハジラミは別名**羽虫**と呼ばれ、鳥類との関連が密接な外部寄生虫。13種があり日本では9種類が確認。**翅がなく**、大部分は鳥類の体表に限って棲息し、体から離れると短時間しか生存できない。**幼虫は2週間ほどで成虫**となり、産卵数が多く**1対の成虫から1ヵ月以内に1万匹以上に増殖**することがある。
- **原因**: ニワトリに寄生するハジラミ5種類:
  - ニワトリオオハジラミ
  - ニワトリハジラミ
  - ニワトリナガハジラミ
  - ニワトリヒメハジラミ
  - カクアゴハジラミ

  **吸血性はないが**、アゴで羽毛・表皮・皮脂などを食し、傷から滲み出た体液を摂取。
- **症状**: 吸血性がないが、不快感やストレスにより、激しい例では**羽毛を喰いちぎるために裸状になる**ことがある。傷からの感染による皮膚疾患もみられる。
- **診断**: 虫体をセロハンテープで捕獲し、スライドグラスに張って観察。
- **治療**: トリサシダニ・ワクモと同じ。

#### ヒゼンダニによる皮膚疾病

- **発生**: 日本のニワトリに寄生する2種類のダニで**疥癬を起こす**。両方とも春から夏にかけて活動が盛ん。
- **原因**:
  - **ニワトリカイセンダニ**（*Knemidocoptes laevis gallinae*）: 翼・尾を除く羽毛の基部付近の皮膚に寄生し、**特に毛根部にトンネルを作る**。その部分に炎症を起こし、ニワトリが自身で毛を抜いたりつついたりすることで疥癬が形成される。雌の体長約0.3mm。
  - **ニワトリアシカイセンダニ**（*K. mutans*）: **脚鱗の下や稀に肉冠**に寄生、皮下組織にトンネルを作り、漿液が分泌されて痂皮を形成。雌の体長約0.5mm。
- **症状**:
  - ニワトリカイセンダニ寄生では皮膚が痂皮を形成し、**細菌の二次感染**を受ける。
  - ニワトリアシカイセンダニ寄生は痂皮をつくり、**重傷例では脚の著しい変形や歩行困難**。慢性症では循環器や腎臓にも障害。
- **診断**: 患部の組織をかき取り**10% KOH液**を加え鏡検によりダニを確認。
- **治療**: **イベルメクチン注射**（0.2〜20mg/kg）、駆虫剤の薬浴、二次感染症の治療。

### 12-5. その他の皮膚疾病（栄養性）

栄養関係では**パントテン酸、ビオチン、ニコチン酸の欠乏**によって皮膚炎が起こる。

## 13. 生殖器疾病

雄の生殖器疾病は少ない。**雌の生殖器疾病は、成鳥初期は産卵による事故で発症することが多く、老齢になると腫瘍が多い**。

### 13-1. 卵墜症

- **発生**: 卵黄物質が**腹腔に落下（卵墜）**し、腹膜の炎症が急激に発生、経過が長いと腹水が溜まる。
- **原因**: 産卵期の鳥が**驚愕**したり、**大腸菌症その他高熱を発する疾病の初期**に発生。
- **症状**: 肉冠のチアノーゼ、食欲廃絶、**腹水のためペンギン様の姿勢**をとる。死亡例も多い。
- **診断**: 腹部の触診（波動感）、腫瘍との区別（削痩は腫瘍の可能性）、レントゲン撮影。
- **治療**: 腹水除去、腹膜炎の処置。

### 13-2. 卵秘症

- **原因**: 産卵開始時期などに**大き過ぎる卵を輸卵管内や総排泄腔内から排出できない**（小型のニワトリに多い）。または大腸菌症のため卵管の閉塞による卵管炎に続発。
- **症状**: 初期は産卵姿勢をとったり不安な行動。持続すれば削痩。
- **診断**: 触診およびレントゲン撮影。
- **治療**: **潤滑剤利用により前後反転摘出**、または破壊除去。**大腸菌症の例は予後不良**。

### 13-3. 非感染性腫瘍

ニワトリの生殖器の腫瘍は、**老齢になると発生頻度が高い**。重要な3つの腫瘍:

1. **卵巣癌**
2. **卵管腺癌**
3. **卵管靭帯の平滑筋腫**

病因はすべて不明であり、**治療法もない**。

## 14. 泌尿器の疾病

鳥類の泌尿器は左右一対の腎臓と尿管からなり、膀胱・尿道はない（§2-5参照）。そのため泌尿器の疾病は腎臓と尿管の異常。

### 14-1. 尿酸沈着症（痛風）

- **発生**: 鳥類は尿中の尿酸塩濃度が高く、**腎臓から直接尿酸塩の形で排泄**するため腎臓の障害を起こしやすい。腎臓・尿管の異常で代謝障害等が生じると**血液中の尿酸塩濃度が高まり、腎臓・肝臓・心臓・腸・腹膜等の漿膜面および関節腔内に沈着**して致死的経過をとる。
- **原因**: 感染症、ヒナの時の寒冷、不適切飼料（**カルシウムやタンパク質のバランス不良**）、**サルファ剤中毒**など。
- **症状**: 元気消失、食欲減退〜廃絶、肉冠の萎縮、**水様・白色下痢便（ザラザラする）**、肛門周囲の白色下痢便の付着。**関節の腫脹や跛行・起立不能**もみられる。
- **診断**: 肉眼所見、**ザラザラした便は特徴的**。尿酸塩の定性試験。
- **治療**: 特になし。

### 14-2. その他の泌尿器疾病

気管支炎の腎炎タイプによる腎炎、大腸菌による尿細管の変性、腎炎ウイルスによる疾病（主にブロイラー）など。

## 15. その他の疾病

### 15-1. ヒナ白痢

- **発生**: 保菌鶏の卵から孵化したヒナが**孵化直後から2〜3日以内に特に症状を示さないで急死**する伝染病。成鳥は無症状だが**卵巣などに保菌して介卵感染**で広範囲に発生。最近は輸入検疫で発見されることが多い。**家畜伝染病予防法における法定伝染病**。
- **原因**: **ヒナ白痢菌（*Salmonella* Pullorum）** の感染による。主に親鳥からの介卵感染。
- **症状**: 孵化直後から特に症状を示さずに死ぬケース（2〜3日齢以内）。または元気・食欲減退、羽毛屹立、**白色で粘り気のある便**を排泄、10日齢をピークとして発症・死亡。**尻汚れや肺炎を伴って呼吸器症状**を示すことも。関節炎・眼球炎を呈するものもある。
- **診断**: 細菌検査または平板急速凝集反応（千葉県血清研究所製）による。
- **治療**: 特になし。**予防的には保菌鶏を淘汰**する。

### 15-2. マレック病（MD）

- **発生**: ヘルペスウイルスに属する**MDウイルスの気道感染**によって**T細胞が腫瘍性増殖**し、内臓や末梢神経に腫瘍を形成。**40〜50日齢を過ぎるニワトリはほとんど感染している**といわれる。感染しても必ずしも発症しない。
  - **定型的MD**: 3〜5ヵ月齢に多発
  - **急性型**: 2〜4ヵ月齢に多発、時に1ヵ月齢未満の発生も
- **原因**: MDウイルス（MDV）は**皮膚の羽包上皮細胞で感染性ウイルスが作られ感染源**となる。このウイルスは**皮膚のフケに存在し、空気伝染**する。生まれたばかりのヒナが最も感受性が高い。
- **症状**:
  - **定型的MD**（末梢神経が侵される）: 脚弱・脚麻痺、起立不能、翼下垂、斜頸
  - **急性型**（主に内臓に腫瘍形成）: 貧血、削痩、緑便排出、神経症状を呈し斃死。死亡率10〜50%。
  - **皮膚型**: 皮膚に腫瘤を形成
  - **眼型**: 瞳孔・虹彩に病変を形成
- **診断**: 病理組織学的検査（**リンパ性白血病との類症鑑別**を含む）、剖検（内臓諸臓器、皮膚、筋肉の腫瘍病変と末梢神経の変化）。抗体検出はほとんどのニワトリが感染しているため診断には不向き。
- **治療**: なし。
- **予防**: ワクチンの接種（産業的飼育以外は実用性に乏しい）。

### 15-3. リンパ性白血病（LL）

- **発生**: トリ白血病ウイルスによるニワトリの**B（細胞）リンパ腫**で、**成鳥になってから好発**。古くから世界各地でみられその発生率は少ないが、20%を超すとの報告もある。
- **原因**: トリ白血病ウイルス（ALV）の感染は、**母鶏から介卵感染する場合**と、**介卵感染したヒナから同時に孵化した未感染ヒナに同居感染する**2つがある。介卵感染したヒナや孵化後短時間に感染したヒナは特殊な免疫状態になり、**ウイルスを終生にわたり排出し続け感染源**になる。ALV起因疾病にはLLのほか、赤芽球症、骨髄芽球症、骨髄球腫症、繊維肉腫、血管腫、腎芽細胞腫、骨化石症などがある。
- **症状**: LLでは食欲減退、削痩、肉冠の萎縮・貧血、緑便下痢便の排出など。**骨化石症では中足骨が腫れて長靴を履いたような特徴的な状態**になる。
- **診断**: LLの生前診断は4ヵ月齢以降で上記症状があり、**肛門から指を挿入してF嚢や肝臓の腫大を触知**できることが多い。斃死例では病理組織学的検査による。**MDとの類症鑑別が必要**。ALVの検出・抗体検出は診断には不向き。
- **治療**: なし。

### 15-4. ロイコチトゾーン症

- **発生**: 胞子虫類に属する住血原虫（*Leucocytozoon caulleryi*）による疾病。主に**吸血性昆虫のニワトリヌカカ**によって媒介される。**発生はニワトリヌカカの活動期である7〜9月にかけて多発、沖縄県では1年中**。
- **原因**: ニワトリヌカカの唾液腺に含まれたスポロゾイトがニワトリ体内に注入されて感染が成立。原虫は**血管に寄生して大きなシゾントになり出血の原因**になる。さらに赤血球に寄生してメロゾイトからガメトサイトに発育し、この過程で赤血球の破壊が起こり**貧血・緑色便排泄の原因**となる。このガメトサイトを含む血液がニワトリヌカカに吸血されれば感染源。
- **症状**: 感染して2週間は無症状。濃感染した例では感染2週間目に**突然の喀血**、うずくまって斃死。生き残ったものでは貧血、産卵停止、緑色便、削痩。
- **診断**: 発生が夏期、臨床症状および剖検所見（皮下、筋肉、全身各臓器の大小の点状出血、腹腔内・気管内・嗉嚢内に血液）。寄生虫学的診断として出血部位からのシゾント検出（生鮮標本、組織切片）、貧血しているニワトリから末梢血液塗抹標本のギムザ染色によりメロゾイトあるいはガメトサイトの検出、**市販の診断液による寒天ゲル内沈降反応**による。
  - **本症と類症鑑別が必要な疾病**: 鶏マラリア（*Plasmodium gallinaceum* と *P. juxtanucleare* の2種、日本では後者）。時に貧血（やや黄色みを帯びる）、クリーム様緑色便。発生は夏〜秋、主に**アカイエカにより媒介**。血液塗抹標本の鏡検で区別できる。
- **治療**: 有効な治療法はない。
- **予防**: **サルファ剤を間歇投与**（5日投与・9日休薬を繰り返す、夏期）。

### 15-5. インフルエンザ（家禽ペスト）

- **発生**: オルソミクソウイルス科の**A型インフルエンザウイルス**による伝染病。ウイルス株により症状・致死率が異なる。**急性で感染率・致死率の高いものを家禽ペストと呼ぶ**。家禽ペストは家畜伝染病予防法における**法定伝染病**に指定。**日本は大正14年（1925）以降発生はない**。近年、香港をはじめとする諸外国で発生があり注意が必要。
- **原因**: 家禽ペストを起こすのは**H5またはH7型のHAをもつウイルス**だが、H5・H7全てが強毒とは限らない。**弱毒ウイルスが強毒に変異**することも知られている。**カモが主要なウイルス保有者・運搬者**と考えられている。カモは感染しても症状は示さないが、腸管でウイルスが増殖。
- **症状**: 肉冠・肉垂・顔面・脚皮膚の**浮腫とチアノーゼ**が共通症状とされるが、ウイルス株・混合感染・環境により様々。元気消失、食欲・飲水欲減退、衰弱、咳、くしゃみ、ラッセル呼吸音、流涙、羽毛屹立、神経症状ならびに下痢のいずれか/組合せ。
- **診断**: ウイルスの分離と同定。**ニューカッスル病との鑑別**。

> **編集者注**: 本書は2004年発行のため、2004年以降の高病原性鳥インフルエンザ（HPAI）の情報は反映されていない。HPAIに関する最新情報は、農林水産省「高病原性鳥インフルエンザ防疫指針」（既取込）を参照のこと。

### 15-6. 家禽コレラ

- **発生**: **パスツレラ・マルトシダ**（*Pasteurella multocida*）感染による細菌性急性敗血症。高い死亡率を示すときは家禽コレラとして**家畜法定伝染病**に指定。日本では昭和51年（1976）にタイから輸入されたキュウカンチョウに集団発生があった。昭和51年以降、種々の鳥類で発生。
- **原因**: *P. multocida*の感受性はニワトリよりも**七面鳥・水禽類・野鳥が高く**、ヒナより**成鳥の発生が多い**。本菌の体内侵入経路は通常呼吸器粘膜だが、他の粘膜や皮膚の創傷からの感染もある。**水禽類では池や沼の汚染で大発生**。
- **症状**:
  - **急性型**: 沈鬱、食欲絶廃、羽毛屹立、口からの粘液漏出、下痢、呼吸速迫を呈し、死亡直前に肉冠・肉垂にチアノーゼ。
  - **慢性型**: 肉垂・眼窩下洞・脚や翼の関節・足蹠・胸骨の粘液嚢に腫脹。結膜・咽頭に滲出性病変、時に斜頸。気管のラッセル・呼吸困難で衰弱・死亡することも。
- **診断**: 細菌検査により病原体を検出。
- **治療**: **本病の治療は保菌鳥をつくるおそれがあるため、治療をせずに殺処分が望ましい**。ケースによってはペニシリン、テトラサイクリン、ニューキノロン系などの抗菌剤を利用。
